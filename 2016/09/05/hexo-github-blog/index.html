一直想有个地方可以记录自己平时的所见所得，发现以前还是太年轻了，以为维护成本很高，没有尝试就放弃了，看了同事和公司的博客，于是决定自己动手也搞一个玩玩。

hexo是一款基于Node.js的静态博客框架gitHub链接地址，如果你想了解更多的话可以直接看代码，下面简单说说搭建我这个博客的步骤吧。

环境配置

  • 安装Node，hexo是基于Node.js的。直接到官网下载最新的安装包安装即可。
  • 安装Git,一般MAC电脑自带安装。
  • 拥有一个GitHub账号（这个没有的话就不要看下面的了吧。。。）
